# RaceMP_F4Style

Enhanced RaceMP Server Resource for BeamMP

Improved version of RaceMP with Formation Lap Free Practice Qualifying sessions and a modern F4-style leaderboard UI.

## New Features

- Formation Lap support (`/formation`)
- Free Practice / Essais Libres (`/el`)
- Qualifying session with timer (`/qualifs`)
- Beautiful modern F4-style leaderboard
- Lap counter (Current / Total)
- Best Lap + Last Lap display
- Gap to leader
- Penalty system visibility
- Real-time improved ranking

## Installation (BeamMP Server)

1. Download the repository
2. Extract the archive
3. Copy the Resources folder into your BeamMP server directory
4. Restart your server

## Admin Commands

| Command                    | Description                                      |
|---------------------------|--------------------------------------------------|
| `/el`                     | Start Free Practice (Essais Libres)              |
| `/qualifs [minutes]`      | Start Qualifying (example: `/qualifs 10`)        |
| `/formation`              | Enable Formation Lap                             |
| `/start`                  | Start the Race                                   |
| `/set laps=12`            | Set number of laps                               |
| `/list`                   | List available track layouts                     |
| `/set track=Track_Name`   | Set the track                                    |
| `/set raceName=Race_Name` | Set the name of the race                         |
